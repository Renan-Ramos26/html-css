Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Páragrafo
  • Títulos
    •
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
    1. Tipos 1, A, a, I, i
    2. Começar pelo meio
  • Listas com demarcadores
    1. disc, circle e square
  • Item individual dentro de uma lista
  • Adicionar Favicon
  • Quebra de linha
  • itálico / Ênfase
    •
  • Texto forte / Negrito
  • Marca Texto
  • Texto pequeno
  • Texto excluído
  • Texto inserido
  • Texto sobrescrito
  • Texto subscrito
  • Trecho de código
  • Texto pré-formatado
  • Âncora / Hiperlink
    1. URL Vinculado
    2. Indicar o idioma da página vinculada
    3. Nova aba _blank / Mesma aba _self
    4. Define a relação do documento atual e o vinculado. (extenal, nofollow. Next, prev, author)
    5. Baixar arquivo
    6. Tipo de arquivo (application/pdf, image/jpeg).
  • Listas de definições
    1. Termo / Sigla
    2. Definição ou Descrição / Significado
  • Imagem dinâmica
    1. Use a tag img e coloque a maior imagem
    2. Depois use a tag source media type.
    3. Em media= coloque max-width: 1050 px
    4. Coloque as imagens na ordem g, m, p.
    5. Type coloque depois /png ou oque for.
  • Áudios
    1. Use o preload= metadata (carrega informações básicas). Não recomenda-se o auto pq tem que carregar o áudio completo.
    2. Use o Controls para aparecer o controle
    3. O auto play só funciona se o PRELOAD= estiver em AUTO.
    4. Use varios tipos de áudios como MP3, OGG e WAV com a TAG Source:src
  • Vídeos
    1. Use a tag video
    2. Width = 650 é o tamanho do video
    3. poster = endereço da foto é o poster do video
    4. controls são os controles do video
  • Vídeos do YouTube ou Vimeo
      • Vai no video online e clica em incorporar, depois é so colar no código.
